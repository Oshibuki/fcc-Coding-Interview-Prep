## 数据结构：扩展运算符既可以用于数组，也可以用于集合。

你是否还记得 ES6 中的`...`操作符？

`...`接收 ES6 中的可迭代对象，并将其转换为数组。

现在我们来创建一个 Set 并检验`...`操作符：

> var set = new Set([1,2,3]);
> var setToArr = [...set]
> console.log(setToArr) // 返回 [ 1, 2, 3 ]
